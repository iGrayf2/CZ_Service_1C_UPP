# Проверка HTTP из 1С

Цель: проверить, что 1С может обратиться к Python-сервису и получить ответ.

## Схема

```text
1С УПП -> HTTP -> Python FastAPI -> JSON-ответ -> 1С УПП
```

## Адрес для теста

Если 1С и Python запущены на одном компьютере:

```text
http://127.0.0.1:8000/ping
```

Если Python-сервис запущен на отдельном компьютере в сети, нужно использовать IP этого компьютера:

```text
http://192.168.x.x:8000/ping
```

## Код для 1С

```1c
Попытка
    Соединение = Новый HTTPСоединение("127.0.0.1", 8000);
    Запрос = Новый HTTPЗапрос("/ping");

    Ответ = Соединение.Получить(Запрос);

    Сообщить("Код ответа: " + Ответ.КодСостояния);
    Сообщить(Ответ.ПолучитьТелоКакСтроку());

Исключение
    Сообщить("Ошибка: " + ОписаниеОшибки());
КонецПопытки;
```

Ожидаемый результат:

```text
Код ответа: 200
```
